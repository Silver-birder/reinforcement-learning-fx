# Q-learningのサンプル

## 概要

強化学習の学習用に実装してみた、   
Gymを利用したQ-learningの動作サンプルです。

## 動作方法

```
python3 cartpole.py
```

## 動作確認

- 1000 Episode前後で 下記のような結果が表示されれば、学習が完了

    ```
    905 Episode finished after 200.000000 time steps / mean 194.540000
    Episode 905 train agent successfuly!
    ```
